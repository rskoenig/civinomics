Civinomics Formatting Guide

This guide shows you how to add formatting to your item. Remember, you can edit your item after submitting it, so don't be afraid to experiment.

Headers

Using hash characters to make H1, H2, H3, H4, H5 or H6 headers: 
# H1
## H2
### H3
#### H4
##### H5
###### H6

will render as:

H1

H2

H3

H4

H5
H6

Links

Adding links with text like this:

[Visit Civinomics Today](http://civinomics.com)
will render like this:
Visit Civinomics Today

Text

Using asterisks or underlines to italicize text like this:

You can emphasize text like *this* or like _this_
will render like this:
You can emphasize text like this or like this

Using double asterisks or double underlines to embolden text like this:

You can make text bold like **this** or like __this__
will render like this:
You can make text bold like this or like this

Blank lines separate paragraphs.

Quoting

Blockquotes are handy to emulate a reply to text, like your email app does. Using a right angle bracket/greater than sign to add block quotes like this:
> This is original text to which I want to respond
will render like this:
This is original text to which I want to respond

Literal text

Using backquotes to mark inline literal text like this:
This is `literal text` and I like it
will render like this:

This is literal text and I like it

You can apply this to entire paragraphs by starting each line with four spaces.

Lists

Using asterisks to make bullet lists like this:

* Asterisks
* make
* bullet
* lists

will render like this:
  • Asterisks
  • make
  • bullet
  • lists
Using numbers to make numbered lists like this:

1. numbers
2. make
3. numbered
4. lists
1. the number used does not matter

will render like this:

  1. numbers
  2. make
  3. numbered
  4. lists
  5. the number used does not matter

Images

Adding inline images hosted elsewhere, with alt text, like this:

![The Civinomics Logo](https://civinomics.com/images/logo.png "Civinomics Logo")

will render like this:

Note that images are not supported in comments.
